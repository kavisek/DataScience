# <center> Tableau Custom Palettes </center>


### Creating Color Paletee Tag

```html
<color-palette name="My Categorical Palette" type="regular" >
<color>#feebe2</color>
<color>#fcc5c0</color>
<color>#fa9fb5</color>
<color>#f768a1</color>
<color>#dd3497</color>
<color>#ae017e</color>
<color>#7a0177</color>
</color-palette>
```

Locate the preference files within the your Tableau Directory

The final preference file 

```html
<?xml version='1.0'?>

<workbook>
  <preferences>
    <color-palette name="My Categorical Palette" type="regular" >
    <color>#feebe2</color>
    <color>#fcc5c0</color>
    <color>#fa9fb5</color>
    <color>#f768a1</color>
    <color>#dd3497</color>
    <color>#ae017e</color>
    <color>#7a0177</color>
    </color-palette>
  </preferences>
</workbook>

```
