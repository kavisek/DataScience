### <center> 10x Data Science with Linux </center>

##### Basic Navigation Sequence

```bash
# Creating and removing directories and filesr
mkdir testing
cd testing
touch file
rm file
cd ..
rmdir -i testing
```

##### Copying

```bash
# Copying files and directories
mkdir testing
cd testing
touch file.txt
cp file.txt file2.txt
cd ..
cp mkdir/* mkdir2/*
```
